# 💎 Diamond Quality and Pricing Analysis (Power BI)

## 📊 Project Overview
This project focuses on analyzing the relationship between diamond quality attributes and their prices using Power BI. The dashboard provides interactive visualizations to explore how factors such as carat, cut, color, and clarity influence diamond pricing.

## 🎯 Objectives
- Analyze the distribution of diamond prices.
- Understand how carat weight affects diamond price.
- Compare price variations across cut, color, and clarity.
- Create interactive dashboards for better data exploration.

## 🛠 Tools Used
- Power BI Desktop
- Power Query (Data Cleaning & Transformation)

## 📁 Dataset Features
The dataset includes important attributes of diamonds:

- Carat – Weight of the diamond  
- Cut – Quality of the cut  
- Color – Diamond color grade  
- Clarity – Diamond clarity level  
- Depth – Diamond depth percentage  
- Table – Width of the diamond top  
- Price – Price of the diamond

## 📊 Dashboard Features
- Price Distribution Analysis
- Carat vs Price Relationship
- Cut Quality Comparison
- Color and Clarity Impact on Price
- Interactive Filters for Data Exploration

## 🔍 Key Insights
- Diamond price increases significantly as carat weight increases.
- Diamonds with better cut quality generally have higher prices.
- Clarity and color grades influence overall diamond pricing.
- Interactive visualizations help in understanding diamond market trends.

## 📌 Conclusion
This Power BI dashboard demonstrates how data visualization can simplify complex datasets and reveal meaningful insights. It helps users understand the key factors influencing diamond pricing.
